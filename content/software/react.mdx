## React 19.2.0 Overview

React 19.2 introduces View Transitions, `useEffectEvent`, `<Activity/>`, and stable Server Components.

### Key Features
- **Server Components** - Render on server, reduce client bundle
- **View Transitions** - Animate elements that update inside a Transition or navigation
- **useEffectEvent** - Extract non-reactive logic from Effects into reusable Effect Event functions
- **Activity Component** - Render "background activity" by hiding UI while maintaining state
- **useActionState** - Manage form state with actions
- **useOptimistic** - Optimistic UI updates
- **React Compiler** - Automatic memoization (stable in Next.js 16)

## Server Components

```typescript
// Server Component (default in Next.js 16)
export default async function ProductList() {
  const products = await fetch('https://api.example.com/products')
  const data = await products.json()

  return (
    <ul>
      {data.map(product => (
        <li key={product.id}>{product.name}</li>
      ))}
    </ul>
  )
}
```

## Client Components

```typescript
'use client'

import { useState } from 'react'

export default function AddToCart({ productId }: { productId: string }) {
  const [count, setCount] = useState(0)

  return (
    <button onClick={() => setCount(count + 1)}>
      Add to Cart ({count})
    </button>
  )
}
```

## What Breaks in Production

### Issue: Hooks in Server Components

**Symptom:** `useState` or `useEffect` in server component causes error

**Fix:** Add `'use client'` directive
```typescript
'use client'

import { useState } from 'react'

export default function MyComponent() {
  const [state, setState] = useState(0)
  return <div>{state}</div>
}
```

### Issue: Async Client Components

**Symptom:** Cannot use `async` keyword with `'use client'`

**Fix:** Use `useEffect` for async operations
```typescript
'use client'

import { useEffect, useState } from 'react'

export default function DataComponent() {
  const [data, setData] = useState(null)

  useEffect(() => {
    fetch('/api/data')
      .then(res => res.json())
      .then(setData)
  }, [])

  return <div>{data?.title}</div>
}
```
